These are the NamedCommands registered with PathPlanner. Use these exact names (case-sensitive) in .auto and .path files. Commands marked WAITS block the auto sequence until finished; INSTANT commands finish immediately and the next step runs right away.

🫳 Intake Commands

Command Name Ends Subsystems Description
PrepareForIntake WAITS Intake Moves intake arm to INTAKE position. Finishes when arm reaches target angle. Does not start rollers.
RunIntake WAITS Intake Extends intake arm to INTAKE position, then starts rollers at full speed once arm is in position. Arm & rollers keep running after command ends.
StopIntake WAITS Intake Stops intake rollers and retracts arm to STOWED position. Finishes when arm reaches stowed angle.
ExtendIntakeArm WAITS Intake Moves only the pivot arm to INTAKE position (no rollers). Use when you need the arm out of the way before a pickup or shot.
RetractIntakeArm WAITS Intake Moves only the pivot arm to STOWED position (does not stop rollers). Pair with StopIntakeRollers if rollers should also stop.
StartIntakeRollers INSTANT Intake Turns on intake rollers at INTAKE speed and finishes immediately. Safety: blocked if arm is in STOWED position.
StopIntakeRollers INSTANT Intake Stops intake rollers and finishes immediately. Arm position is unchanged.

🔫 Shooting Commands

Command Name Ends Subsystems Description
PrepareToFire WAITS Drivetrain Aims at hub AprilTag using Limelight TX with PD control. Rotates drivetrain toward target. Auto-ends when aimed (TX ≤ 3°) or on timeout. Uses tuned gains (kP=0.03, kD=0.005).
Fire WAITS Feeder, Shooter, Hood, Floor Spins up shooter & positions hood using interpolation table based on Limelight distance (fallback 3 m). Once at speed, engages feeder & floor for a fixed duration, then auto-ends.

🧗 Climb Commands

Command Name Ends Subsystems Description
PrepareToClimb WAITS Hanger Extends hanger to EXTEND_HOPPER position to prepare for climbing. Finishes when hanger reaches target extension.
PrepareToClimbLeft WAITS Hanger Same as PrepareToClimb but configured for a left-side approach to the tower.
PrepareToClimbRight WAITS Hanger Same as PrepareToClimb but configured for a right-side approach to the tower.
Climb WAITS Hanger Executes full climb sequence: moves hanger to HANGING position, then to HUNG position. Finishes when fully hung.
